アサリとブロッコリーのワイン蒸し

調理時間5分

エネルギー(1人分)66kcal

食塩相当量(1人分)0.7g

日清やさし～く香るエキストラバージンオリーブオイル 350gPET

材料 （4人分）

  • アサリ(砂抜き)　300g
  • ブロッコリー　1個
  • にんにく　1片
  • 赤唐辛子　1本
  • 白ワイン　1/3カップ
  • 日清やさし～く香るエキストラバージンオリーブオイル　大さじ1
  • こしょう　少々

作り方

  • ❶アサリは洗って水けをきる。ブロッコリーは小さめの房に分ける。にんにくは半分にして潰す。赤唐辛子は種を除く。
  • ❷フライパンに日清やさし～く香るエキストラバージンオリーブオイル、にんにく、赤唐辛子を入れて弱火にかけ、香りがでてきたらアサリ、ブロッコリーとワインを入れてふたをして蒸し、アサリが開いたらこしょうを振る。
