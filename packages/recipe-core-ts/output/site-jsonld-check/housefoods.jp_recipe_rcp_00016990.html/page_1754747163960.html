調理時間 約20分
エネルギー 410kcal
食塩相当量 2.2g

エネルギー・食塩相当量は1人分の値です。

４０ｇ　特選本香り生しょうが 小さじ２
酒 大さじ１・1/2
砂糖 大さじ１・1/2
しょう油 大さじ１・1/2
みりん 大さじ１
水 大さじ１

作り方

  1. 小さめのボウルに（ａ）を入れ混ぜる。
  2. 豚肉は食べやすい大きさに切り、薄力粉を薄くまぶしておく。玉ねぎは７～８ｍｍ幅のくし切りにする。
  3. フライパンにサラダ油を熱し、（２）の豚肉を重ならないように焼き、豚肉に焼き色がつき、ある程度火が通ったらいったん取り出す。
  4. 同じフライパンで（２）の玉ねぎをしんなりするまで炒める。
  5. （３）の豚肉をもどし、（１）を加えて煮立たせ、少し煮詰める。

一口メモ

＊豚肉は小間切れ肉でもよいです。
＊豚肉に薄力粉をまぶすことで、柔らかく仕上がります。
